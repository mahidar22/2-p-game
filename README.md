# 🎮 2 Player Mini-Games App

A complete collection of 70 mini-games for 2 players.

## 🚀 Quick Start

```bash
npm install
npm start
```

## 📱 Features

- 70+ Mini-Games
- Local Multiplayer
- Bot AI Support
- Tournament Mode
- Beautiful UI/UX

## 🎯 Game Categories

- Arcade (12 games)
- Board Games (14 games)
- Sports (12 games)
- Action (16 games)
- Puzzle (8 games)
- Casual (8 games)
